# Student Record Management System

A modern, responsive PHP application for managing student records.

## Features
- **Create**: Add new student records with validation.
- **Read**: View all students in a clean, modern dashboard.
- **Update**: Edit existing student information.
- **Delete**: Safely remove records with a confirmation prompt.
- **Premium UI**: Built with a custom CSS design system using the Inter font family.

## Tech Stack
- PHP
- MySQL (PDO)
- Vanilla CSS
- Google Fonts

## Implementation Details
- `index.php`: Dashboard showing all records.
- `create.php`: Form to add new students.
- `edit.php`: Form to modify existing records.
- `delete.php`: Backend logic for record removal.
- `db.php`: Database connection using PDO.
- `style.css`: Custom design tokens and modern layouts.
